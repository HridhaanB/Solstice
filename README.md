# Solstice
Solstice is a simple dashboard website that allows you to view all the important information in your Mars base at one simple location.
## How to run
Run app.py to host the website at port 5000
## Features
In the dashboard, you can see
- Visibility
- Oxygen Levels
- Water Usage
- Surface Temperature
- Messages from Crewmates
- Alerts and Notifications
## Why Use Solstice  
Solstice is the best at centralizing vital information for your needs. There is nothing else quite like it.
